# This is a simulation of a basic ecosystem
  - This includes:
    - Bacteria
    - Virus
    - Food Pellets
   
  - Bacteria eats the food pellets
  - Virus eats the bacteria
  - Both bacteria and virus need to eat in order to stay alive
  - Both lose health over time
    - They also lose more health when they move
